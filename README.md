# Personal Trainer Dashboard

This is a client management dashboard generated from personal trainer data.

## About

This dashboard was automatically generated using OpenClaw and displays client information from three databases:
- Client information
- Scheduling data
- Nutrition plans and progress

## Structure

- `index.html` - Main dashboard showing all clients
- `client_X.html` - Individual client pages (where X is the client ID)

## Deployment

This site is designed to be hosted on GitHub Pages.
